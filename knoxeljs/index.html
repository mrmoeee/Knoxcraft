Knoxcraft brings programming 3D Logo to life using Minecraft-themed visualizations. Write code in Java, Python, or Scratch (Blockly), and visualize your voxel-based creations, through Minecraft or in your web browser!

Knoxcraft comes in two flavors:

  • Knoxcraft: A Minecraft mod for Spongepowered, a Forge-compliant modding API
  • KnoxelJS: A voxel-based environent that runs in your web browser using WebG, VoxelJS, and Sphax textures
